Bienvenido al Sistema de Pronóstico Hidrométrico

Este sistema permite consultar el nivel de riesgo de inundación y sequía en la Ciudad de México y el Estado de México, a partir de los datos meteorológicos proporcionados por CONAGUA.

Selecciona una de las opciones del menú para visualizar las predicciones detalladas.

Proyecto académico desarrollado para fines educativos.

    Patrones Climáticos:

  • Templado subhúmedo (C(w)): Cubre la mayor parte del estado, con temperaturas moderadas y lluvias principalmente en verano
  • Cálido subhúmedo (A(w)): Se encuentra en el sur, con temperaturas más altas y lluvias más abundantes en verano
  • Semiárido templado (B(s)): Presente en el norte, con clima seco y temperaturas moderadas
  • Frío (E(T)H) y semifrío (C(E)): Se localizan en zonas altas y montañosas, con temperaturas bajas y heladas frecuentes

    Herramientas Utilizadas:

    • Python 3.13
    • FastAPI (para crear la API)
    • Uvicorn (servidor para la API)
    • Pandas (análisis de datos)
    • Visual Studio Code
    • HTML, CSS y JavaScript (para la interfaz web)

    Funcionamiento General:

    1. El usuario selecciona CDMX o EDOMEX desde la página web.
    2. La página hace una consulta HTTP a la API local de FastAPI.
    3. FastAPI lee los archivos de datos (CONAGUA) con Pandas.
    4. Calcula el riesgo de inundación y sequía.
    5. Regresa los resultados a la web y los muestra al usuario.

    Estructura del Proyecto:

    • Carpeta web:
      • index.html (página principal)
      • mapa_cdmx.png
      • mapa_edomex.png
    • main.py (backend con FastAPI)
    • Archivos CSV de CONAGUA para CDMX y EDOMEX

    Propósito:

    Este proyecto tiene fines educativos y sirve como demostración de cómo integrar datos meteorológicos con predicciones simples a través de una interfaz web conectada a un backend Python.
